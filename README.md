# Kun'ts adventure

Knut's adventure is a game.

Controls:

* ←: Move left

* ↑: Move up

* ↓: Move down

* →: Move right

* r: Restart

 F: Full screen (will restart level)

## This game uses the [Löve](https://love2d.org/) library, you will need to have downloaded it to play the game.
